# 🌐 Manchitra - Smart Navigation for Durga Puja and Beyond

**Manchitra** is a powerful map-based web application that helps users discover and navigate **Durga Puja mandaps** and nearby essentials like restaurants, cafes, petrol pumps, and transit locations. Built using **React**, it offers features similar to Google Maps with a cultural twist.

---

## 🚀 Features

- 🗺️ Interactive map with real-time navigation
- 🎉 Durga Puja mandap locator at various distances
- 🍽️ Nearby restaurants, cafes, and food stalls
- ⛽ Petrol pumps and gas stations
- 🚉 Railway stations & 🚌 bus stops
- 🌙 Dark mode support for better night-time use
- ⚡ Smooth animations and responsive UI
- 📍 Distance-wise filters and dynamic location updates

---

## 🛠️ Tech Stack

- **Frontend**: React.js, Tailwind CSS, Framer Motion
- **Map Integration**: Google Maps API
- **State Management**: React Context API
- **Animations**: Framer Motion
- **Design**: Figma prototype (with dark mode & interactions)
